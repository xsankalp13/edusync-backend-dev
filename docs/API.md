API Index

This document provides a quick index of commonly used APIs in the monolith.

Students
- GET /api/students
- POST /api/students
- GET /api/students/{id}

Authentication
- POST /api/auth/login
- POST /api/auth/refresh

HRMS
- GET /api/hrms/staff
- POST /api/hrms/staff
- POST /api/hrms/staff/{id}/bank-details

Notes
- Where OpenAPI/Swagger is configured, regenerate using the project's OpenAPI plugin (if present). If no plugin is configured, use runtime UI at /swagger-ui.html when running locally.
